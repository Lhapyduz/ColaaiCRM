# Configuração de Ambiente - Cola Aí

## Supabase

Para conectar o aplicativo ao Supabase, crie um arquivo `.env.local` na raiz do projeto com:

```
NEXT_PUBLIC_SUPABASE_URL=sua_url_supabase
NEXT_PUBLIC_SUPABASE_ANON_KEY=sua_anon_key_supabase
```

Você pode encontrar essas credenciais em:
1. Acesse [supabase.com](https://supabase.com)
2. Entre no seu projeto
3. Vá em Settings > API
4. Copie a URL e a anon/public key
